# Udharkhate Next.js

This is a Next.js project converted from a React SPA.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/utils`: Utility functions and storage logic.
- `src/styles`: Global styles and Tailwind CSS configuration.

## Features

- **Next.js App Router**: Uses the latest Next.js routing features.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **LocalStorage**: Data persistence using browser local storage.
